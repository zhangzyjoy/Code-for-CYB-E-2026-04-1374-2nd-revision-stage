This repository contains the simulation and experimental validation code for the revised manuscript with submission ID CYB-E-2026-04-1374.<br/>
This link indicates the proposed control approach for a formation control issue of uncrewed aerial vehicle (UAV) teams, where each quadcopter UAV node is decoupled into rotational and translational subsystems, respectively.<br/> 
In the proposed method, a distributed practical fixed-time distributed formation controller (PFxTDFC) is established for the consensus in translational subsystem.<br/> 
Practical fixed-time distributed state observers (PFxTDSO) are developed to estimate the desired velocity and position for each follower UAV and to maintain fully decentralized realization.<br/> 
Leveraging the logarithmic mapping of rotational errors in Lie algebra space, a nonsingular Lie-algebra-based sliding mode attitude controller (NLSMAC) is developed to attain practical fixed-time singularity-free anti-disturbance attitude tracking in rotational subsystem.<br/>
