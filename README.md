This repository contains the simulation and experimental validation code for the revised manuscript with submission ID CYB-E-2026-04-1374. This link indicates a distributed practical fixed-time distributed formation controller (PFxTDFC) for the cooperative control issue of uncrewed aerial vehicle (UAV) teams. In the proposed method, practical fixed-time distributed state observers (PFxTDSO) are developed to estimate the desired velocity and position for each follower UAV and to maintain fully decentralized realization.
